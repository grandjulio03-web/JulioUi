🚨 Alerts

Dismissible notification components with theme colors and smooth animations.

Overview

Alerts are flexible notification components that can display success, error, warning, or informational messages. They feature dismissible functionality, theme variants, and smooth fade-out animations.

Basic Usage

<div class="alert alert-ace"> <span>Your message here</span> <button class="alert-close">×</button> </div>

Theme Variants

Alerts support all framework theme colors. Simply change the modifier class to match your design.

Component Structure

Alerts use a simple and semantic structure with minimal required classes.

Required Classes

  • .alert - Base container for alert styling
  • .alert-{theme} - Theme color (ace, beta, gamma, delta, epsilon)
  • .alert-close - Dismissible button functionality

JavaScript Behavior

Alerts include automatic dismissible functionality. Click the close button to remove the alert with a smooth fade animation.

Auto-initialization

No JavaScript initialization required. All alerts with .alert-close buttons automatically get dismissible behavior.
