Backgrounds

The feature Bootstrap and Tailwind don't ship. Mesh gradients, aurora, glow, noise, dots, grid, beam — all pure CSS, one class. No JS.

Pattern: air-bg [style] [tone?] — the air-bg class sets position:relative; overflow:hidden. All children need position:relative; z-index:1 to appear above pseudo-element layers.

Mesh — animated blob gradients

Radial gradient blobs that drift slowly. Best for hero sections and landing pages.

Animated mesh gradient

Pure CSS background-size animation

HTML
<div class="air-bg mesh">
  <!-- children need position:relative; z-index:1 -->
  <div style="position:relative; z-index:1;">Content</div>
</div>

Aurora — northern lights

Layered conic gradients with blur, rotating continuously. Best for dark hero sections.

Northern Lights

Conic gradient layers, rotating on loop

Beam — sweeping light (hover to trigger)

Hover me — beam sweeps
Continuous loop (.beam-loop)
HTML

<!-- Continuous loop -->
<div class="air-bg beam gamma beam-loop">...</div>
