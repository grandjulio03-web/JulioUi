DocsForms

Forms

Input, select, textarea, search, toggle, checkbox, radio, range, file upload. All accessible, all toned, all dark-mode aware.

Pattern: air-field wrapper → air-label → air-input [tone] [variant] [size] → air-hint

Field structure

We'll never share your email.
HTML
<div class="air-field">
  <label class="air-label">Email <span>*</span></label>
  <input type="email" class="air-input ace" placeholder="you@example.com">
  <span class="air-hint">We'll never share your email.</span>
</div>

Input variants

HTML
<input class="air-input" placeholder="Default">
<input class="air-input ghost">
<input class="air-input flat">
<input class="air-input neon ace">

<!-- Glass variant for dark backgrounds -->
<input class="air-input glass">

Toggle switch

HTML
<label class="air-toggle ace">
  <input type="checkbox" checked>
  <span class="air-toggle-track">
    <span class="air-toggle-thumb"></span>
  </span>
  Label text
</label>

Checkbox & Radio

HTML
<!-- Checkbox -->
<label class="air-check ace">
  <input type="checkbox" checked>
  <span class="air-check-box"></span>
  Label
</label>

<!-- Radio -->
<label class="air-radio delta">
  <input type="radio" name="group">
  <span class="air-radio-dot"></span>
  Option
</label>

Range slider

HTML
<input type="range" class="air-range ace" min="0" max="100" value="65">

Form layouts

HTML
<!-- Vertical stack -->
<div class="air-form">
  <div class="air-field">...</div>
  <div class="air-field">...</div>
</div>

<!-- Two-column row -->
<div class="air-form-row">
  <div class="air-field">...</div>
  <div class="air-field">...</div>
</div>

<!-- Input + button combo -->
<div class="air-input-action">
  <input class="air-input" placeholder="Subscribe...">
  <button class="air-btn ace">Join</button>
</div>

Class reference

ClassElementDescription
air-fieldwrapperFlex column gap. Wraps label + input + hint.
air-label<label>Form field label. <span>*</span> shows red required marker.
air-input<input>Base input. Add tone for focus ring color.
air-select<select>Styled select with custom arrow.
air-textarea<textarea>Resizable textarea.
ghost flat glass neonon inputsStyle variants.
sm lgon inputsSize variants.
is-error is-successon inputsValidation state classes.
air-hint<span>Helper text below input.
air-error<span>Error message. Add .is-visible to show.
air-toggle<label>Toggle switch. Needs track + thumb inside.
air-check<label>Checkbox. Needs .air-check-box span inside.
air-radio<label>Radio button. Needs .air-radio-dot span inside.
air-range<input type="range">Styled range slider.
air-formwrapperVertical form layout with gap.
air-form-rowwrapperTwo-column row (stacks on mobile).
air-input-actionwrapperInput + button fused inline.
