Animations & Transforms

Three layers: hover transforms, scroll-triggered entry animations (IntersectionObserver), and continuous loop animations. One attribute — zero config.

Hover transforms

Add a class to any element. The transform fires on hover — CSS only, no JS.

.lift
.grow
.tilt
.bright
.dim
.push-right
HTML
<div class="lift">...</div>        translateY(-4px) + shadow
<div class="grow">...</div>        scale(1.04)
<div class="shrink">...</div>      scale(0.97)
<div class="tilt">...</div>        3D perspective rotate
<div class="flip-x">...</div>      scaleX(-1)
<div class="bright">...</div>      filter brightness +12%
<div class="dim">...</div>         opacity 65% on hover
<div class="push-right">...</div>  translateX(4px)
<div class="spin-hover">...</div>  rotate(180deg)

Scroll entry — data-animate

Add data-animate="name" to any element. When it enters the viewport, air-animate.js adds the class .awake and CSS transitions fire.

HTML
<!-- Basic scroll entry -->
<div data-animate="rise">Slides up when scrolled into view</div>

<!-- With delay (ms) -->
<div data-animate="fade" data-delay="200">Fades in after 200ms</div>

<!-- Re-animate on every scroll (default is once) -->
<div data-animate="pop" data-once="false">Pops in every time</div>

<!-- Stagger: auto-delay all children -->
<div data-stagger="rise" data-stagger-step="80">
  <div>Appears first</div>
  <div>80ms later</div>
  <div>160ms later</div>
</div>

Direct CSS animations — plays immediately on page load

HTML
<div class="anim-rise">...</div>     plays on page load
<div class="anim-pop">...</div>      spring scale-in
<div class="anim-blur">...</div>     resolves from blur
<div class="anim-wipe">...</div>     clip-path wipe

<!-- Delay utilities -->
<div class="anim-rise delay-200">...</div>
<div class="anim-rise" data-delay="400">...</div>

<!-- Duration utilities -->
<div class="anim-rise anim-slow">...</div>    slower
<div class="anim-rise anim-fast">...</div>    faster

JS API — air-animate.js

JavaScript
// Re-scan DOM for new data-animate elements
Air.animate.refresh();

// Manually wake an element (trigger animation)
Air.animate.wake(document.querySelector('.my-card'));

// Put element back to sleep (re-animates next scroll)
Air.animate.sleep(document.querySelector('.my-card'));
