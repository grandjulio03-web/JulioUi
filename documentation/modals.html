Modals & Drawers

Open with data-open="id". Close with data-close, Escape key, or backdrop click. Focus trapped, stacking supported, drawers included.

Requires: Include air-modal.js before </body>.

Live demo

Structure

HTML — complete modal
<!-- Trigger -->
<button class="air-btn ace" data-open="my-modal">Open</button>

<!-- Modal -->
<div class="air-modal" id="my-modal">
  <div class="air-modal-box ace">

    <div class="air-modal-head">
      <div>
        <h3 class="air-modal-title">Title</h3>
        <p  class="air-modal-sub">Subtitle</p>
      </div>
      <button data-close class="air-modal-x">&#10005;</button>
    </div>

    <div class="air-modal-body">
      Content goes here...
    </div>

    <div class="air-modal-foot">
      <button data-close class="air-btn ace ghost sm">Cancel</button>
      <button data-close class="air-btn ace sm">Confirm</button>
    </div>

  </div>
</div>

Sizes & variants on air-modal-box

HTML
<!-- Sizes -->
<div class="air-modal-box sm">...</div>    380px

<div class="air-modal-box lg">...</div>    700px
<div class="air-modal-box xl">...</div>    900px
<div class="air-modal-box full">...</div>  full screen

<!-- Tone accent (top border) -->
<div class="air-modal-box ace">...</div>
<div class="air-modal-box beta">...</div>

<!-- Style variants -->
<div class="air-modal-box glass ace">...</div>
<div class="air-modal-box dark ace">...</div>

<!-- Bottom sheet on mobile -->
<div class="air-modal sheet" id="my-sheet">...</div>

<!-- Drawer (right side panel) -->
<div class="air-drawer" id="my-drawer">...</div>
<div class="air-drawer left" id="my-left">...</div>

JavaScript API

JavaScript
Air.modal.open('my-modal')   // open by ID
Air.modal.close('my-modal')  // close by ID
Air.modal.close()             // close topmost open modal

// Subscribe to events
Air.modal.onOpen(function(modalEl)  { console.log('opened', modalEl.id); });
Air.modal.onClose(function(modalEl) { console.log('closed', modalEl.id); });

Class reference

Class / AttributeDescription
air-modalBackdrop + positioner. Hidden by default.
air-modal.sheetBottom-sheet variant (slides from bottom).
air-modal-boxThe dialog card inside air-modal.
sm lg xl fullSize modifiers on air-modal-box.
ace beta gamma deltaTop border accent on air-modal-box.
glass darkStyle variants on air-modal-box.
air-modal-headTop row — title + close button.
air-modal-titleHeading inside air-modal-head.
air-modal-subSubtitle/description inside air-modal-head.
air-modal-xClose (×) button styling.
air-modal-bodyScrollable content area.
air-modal-footBottom row — action buttons. Add .split for space-between.
air-modal-iconColored icon box for confirm dialogs.
air-drawerRight-side slide panel. Add .left for left.
data-open="id"On any element — opens target modal/drawer on click.
data-closeOn any element inside modal — closes parent.
