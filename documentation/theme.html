Light / Dark Theme

One attribute on <html>. Every component switches automatically. System preference detected. Preference saved. Zero flash.

How it works

Air uses CSS custom properties for all colors. Dark mode overrides those properties at [data-theme="dark"] on the root element. All components automatically follow.

CSS mechanism
/* Light mode (default) */
:root {
  --air-white: #ffffff;
  --air-900: #111827;
  --air-200: #e4e8ec;
  /* ... all tokens ... */
}

/* Dark mode — override same tokens */
[data-theme="dark"] {
  --air-white: #111b0f;  /* dark background */
  --air-900: #edf8eb;   /* light text */
  --air-200: #263821;  /* dark borders */
}

/* Components use the tokens — no changes needed */
.air-card {
  background: var(--air-white);  /* auto-switches */
  border: 1px solid var(--air-200); /* auto-switches */
}

Setup

HTML — complete setup
<html lang="en">
<head>
  <!-- 1. Flash prevention — BEFORE the CSS link -->
  <script>
    try {
      var t = localStorage.getItem('air-theme');
      if (t) document.documentElement.setAttribute('data-theme', t);
      else if (window.matchMedia('(prefers-color-scheme: dark)').matches)
        document.documentElement.setAttribute('data-theme', 'dark');
    } catch(e) {}
  </script>

  <!-- 2. CSS -->
  <link rel="stylesheet" href="dist/my-framework.css">
</head>
<body>
  <!-- 3. Toggle button (put anywhere) -->
  <button class="air-theme-btn" data-theme-toggle>
    <svg class="icon-moon">...</svg>  shown in light mode
    <svg class="icon-sun">...</svg>   shown in dark mode
  </button>

  <!-- 4. JS at bottom -->
  <script src="src/js/air-theme.js"></script>
</body>

Toggle button — air-theme-btn

The .air-theme-btn class styles a toggle button. The data-theme-toggle attribute wires it to the JS engine. Put it anywhere — nav, sidebar, settings page.

HTML
<button class="air-theme-btn" data-theme-toggle>
  <!-- Moon icon: shown in light mode -->
  <svg class="icon-moon" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <path d="M21 12.79A9 9 0 1 1 11.21 3 7 7 0 0 0 21 12.79z"/>
  </svg>
  <!-- Sun icon: shown in dark mode -->
  <svg class="icon-sun" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <circle cx="12" cy="12" r="5"/>
    <line x1="12" y1="1" x2="12" y2="3"/> ...rays
  </svg>
</button>

JavaScript API

JavaScript
Air.theme.toggle()           // switch between light and dark
Air.theme.set('dark')         // force dark
Air.theme.set('light')        // force light
Air.theme.get()              // returns 'dark' | 'light'

// Subscribe to theme changes
Air.theme.onChange(function(theme) {
  console.log('Theme is now:', theme);
  // Update charts, third-party widgets, etc.
});
