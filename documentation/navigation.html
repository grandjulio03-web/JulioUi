Navigation

Three archetypes: air-nav (rail), air-nav float (pill), air-nav floor (bottom bar). Plus air-side sidebar and air-trail breadcrumb. Write links once — JS auto-builds the mobile panel.

What makes Air Nav unique: Sliding indicator animates between links in real time. Mobile panel is auto-cloned from your desktop links — no duplication. Scroll behavior (auto-glass) from one data-scroll attribute. Viewport-aware dropdowns flip direction automatically.

Class reference

Class / AttributeElementDescription
air-nav<nav>Sticky rail nav. Default tone: ace.
air-nav float<nav>Fixed pill nav, centered, hovers above content.
air-nav floor<nav>Bottom tab bar.
dark glass clearon air-navVisual style modifier.
data-scrollon air-navAuto-glassmorphism when page is scrolled.
data-indicatoron air-navInject sliding active indicator (underline/pill).
.brandinside air-navLogo / site name.
.linksinside air-navNav links container (no ul/li needed).
.linkinside .linksIndividual nav link. Add .active.
data-fly="id"on .linkOpens dropdown .fly#id on hover/click.
.flyinside .linksDropdown panel. Add .wide for mega.
.endinside air-navRight-side CTA area.
.menuinside air-navHamburger. JS auto-builds mobile panel from links.
.tabinside air-nav.floorBottom bar tab item.
air-side<nav>Sidebar nav with mark/section/path/tail/foot parts.
air-trail<nav>Breadcrumb. Use .crumb + .crumb.active.
