Docs Buttons

Buttons

Chromatic depth shadows, spring physics, 6 variants × 5 tones. Write: air-btn ace glow lg — done.

Pattern: air-btn [tone] [variant] [size] [shape] — each is a separate class, all optional.

Solid — default

HTML
<button class="air-btn ace">Ace</button>
<button class="air-btn beta">Beta</button>
<button class="air-btn gamma">Gamma</button>
<button class="air-btn delta">Delta</button>
<button class="air-btn epsilon">Epsilon</button>

Ghost — outline only, fills on hover

HTML
<button class="air-btn ace ghost">Ghost</button>

Soft — tinted background

HTML
<button class="air-btn ace soft">Soft</button>

Glow — radiant color glow on hover

HTML
<button class="air-btn ace glow">Glow</button>

Glass — frosted, best over colored backgrounds

HTML
<button class="air-btn ace glass">Glass</button>

Gradient — animated gradient flow on hover

HTML
<button class="air-btn ace gradient">Gradient</button>

Sizes

HTML
<button class="air-btn ace xs">XS</button>
<button class="air-btn ace sm">Small</button>
<button class="air-btn ace">Default</button>
<button class="air-btn ace lg">Large</button>
<button class="air-btn ace xl">XL</button>

Shapes — pill, icon

HTML
<!-- Pill -->
<button class="air-btn ace pill">Pill</button>

<!-- Icon-only (perfectly square) -->
<button class="air-btn ace icon" aria-label="Star">★</button>

Block — full width

HTML
<button class="air-btn ace block">Full width</button>

Button group

HTML
<div class="air-btn-group">
  <button class="air-btn ace">Left</button>
  <button class="air-btn ace">Center</button>
  <button class="air-btn ace">Right</button>
</div>

<!-- Add .pill for rounded group -->
<div class="air-btn-group pill">...</div>
