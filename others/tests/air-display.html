Display, Grid & Positioning

Air's own English words for display types. Powerful grid with named layouts. Positioning helpers. All responsive.

Air's display vocabulary — human words over CSS jargon
shelf

display:flex; align-items:center — items sit side by side on a shelf. The most common flex pattern.

pile

display:flex; flex-direction:column — items stacked in a pile.

line

display:inline-flex; align-items:center — inline horizontal row, flows with text.

flow

display:flow-root — creates a new Block Formatting Context. Clears floats.

hide

display:none — gone, takes no space.

blind

visibility:hidden — invisible but still takes space. Keeps layout stable.

ghost

opacity:0; pointer-events:none — invisible, no interaction, still there.

center

display:flex; align-items:center; justify-content:center — dead center both axes.

push

margin-left:auto — pushes item to the far right inside a shelf.

grow

flex-grow:1 — fills remaining space in a flex container.
