Air notes — callout blocks
Tip: Use .air-balance on headings to prevent orphan words and improve line breaks across breakpoints.
Warning: Gradient text uses -webkit-background-clip: text. This won't work on elements with display: inline-block on some browsers.
Note: The .air-display font size is clamp()-based — it scales fluidly from 2.5rem on mobile to 5rem on large screens.
Done: All Air type classes include text-wrap: balance for headings and text-wrap: pretty for body text.

Prose — .air-prose wrapper

Writing for the Web

The best interfaces feel invisible. When typography is right, users read content, not CSS classes. The Air type system is designed to get out of the way.

The Three Pillars

Hierarchy, rhythm, and contrast. Each level of the type scale serves a specific role in the visual hierarchy — not as decoration, but as a communication tool.

Typography is to literature as musical performance is to composition — it brings the written word to life.
