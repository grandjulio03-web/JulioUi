Getting started

Air Framework is a modern CSS system designed for real products. It ships a complete token set, 25+ components, and five lightweight JS engines — all in a single compiled CSS file.

Design is not just what it looks like and feels like. Design is how it works.

Installation

Drop the compiled stylesheet into your <head> and you're done:

<link rel="stylesheet" href="dist/my-framework.css">

Optional JS engines

  • air-modal.js — modals, drawers, and sheets
  • air-fold.js — accordion collapse
  • air-drop.js — dropdown menus
  • air-icon.js — animated SVG icon system

Each engine is self-contained with zero dependencies. Load only what you need.
