Decimal Spacing p-0.1 — p-0.5

Bootstrap uses p-1 through p-5. Air uses p-0.1 through p-0.5 — the same values, with a cleaner decimal notation that reads as a scale, not arbitrary integers.

CSS escaped-dot selectors JS MutationObserver p · m · pt · pb · pl · pr · px · py · gap
How it works — 3 things to know
1. HTML attribute

Dots are valid in class names

HTML allows any character in class="" except spaces. class="p-0.3" is perfectly valid HTML.

2. CSS selector

Dot escaped with backslash

In CSS, dots start class selectors — so the literal dot is escaped: .p-0\.3 { padding: 1rem; }

3. JavaScript layer

Dynamic elements covered

air-spacing.js scans all [class] elements and uses a MutationObserver to catch nodes added after page load.

JavaScript engine — dynamically injected elements
How the JS works: air-spacing.js uses a MutationObserver to watch the DOM. When new nodes appear (React renders, HTMX swaps, JS injection), it scans their classes and applies the decimal spacing via inline styles — zero extra code needed.

Click a button above — JS will inject a div with that class here

The injected elements have no inline styles — air-spacing.js reads their class and applies the padding automatically.
