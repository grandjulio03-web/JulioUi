📖 How to Use

Basic Usage

<div class="mt-1">Top margin 16px</div>
<div class="px-2_5">Horizontal padding 40px</div>

Available Values

Decimal: 0_5, 0_75, 1_25, 1_5, 1_75, 2_25, 2_5

Whole: 1, 2

Range: 0.5rem to 2.5rem (8px to 40px)

Directions

Margin: mt, mr, mb, ml, mx, my

Padding: pt, pr, pb, pl, px, py

Responsive

Tablet: Scale down 25%

Mobile: Scale down 50%

Auto: No extra classes needed
