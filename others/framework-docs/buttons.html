🔘 Buttons

Styled button components with theme colors and hover effects.

Overview

Buttons are interactive elements with smooth hover transitions, theme color support, and multiple size options. Perfect for forms, navigation, and action triggers.

Basic Usage

<button class="btn">Default Button</button> <button class="btn btn-primary">Primary Button</button> <button class="btn btn-ace">Themed Button</button>
